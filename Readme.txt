Render Preset Manager – Blender Add-on
=====================================

Author: Zekeriya AYAN
Contact: zekeriya.ayan@gmail.com
License: GNU GPL v3+

DESCRIPTION
-----------
Render Preset Manager allows you to save your current Blender render
settings as reusable .blend preset files.

Only render-related settings are stored.
No objects, collections or scene data are modified.

SUPPORTED ENGINES
-----------------
• Cycles
• Eevee / Eevee Next

WHAT CAN YOU DO?
----------------
• Save current render settings as a preset
• Quickly re-apply presets to any scene
• Build your own render preset library
• Switch between preview, final and animation setups instantly

WORKFLOW
--------
1. Set up your render settings
2. Click "Quick Export"
3. Save preset as a .blend file
4. Select and apply presets anytime

WHO IS THIS FOR?
----------------
• Blender artists
• Freelancers
• Archviz & product render artists
• Anyone who uses multiple render setups

INSTALLATION
------------
1. Open Blender
2. Edit > Preferences > Add-ons
3. Install ZIP
4. Enable Render Preset Manager
